## Phase 2: Materials Preparation (Weeks 1-4)

- [ ] Update resume -- create direction-specific versions:
  - Data Analyst version: emphasize SQL, Python, statistics, data projects
  - Consulting version: emphasize leadership, problem-solving, healthcare domain
  - PM version: emphasize product thinking, user research, cross-functional work
- [ ] Prepare a federal resume version (2-4 pages, detailed format) if targeting government roles
  - Read OPM guidance: https://www.opm.gov/policy-data-oversight/hiring-information/students-recent-graduates/
- [ ] Get resume reviewed at JHU Imagine Center:
  https://imagine.jhu.edu/
- [ ] Update LinkedIn profile with BME-relevant keywords for your target direction

## Phase 4: Applications -- Government Roles

Government hiring typically takes 2-6 months from application to offer. Apply early.

- [ ] Create a USAJobs account:
  https://www.usajobs.gov/
- [ ] Set up saved searches and email alerts for "data analyst," "data scientist," "management analyst," "program analyst" on USAJobs
- [ ] Monitor NIH jobs:
  https://jobs.nih.gov/
- [ ] Monitor FDA jobs:
  https://www.fda.gov/about-fda/jobs-and-training-fda
- [ ] Monitor CDC jobs:
  https://jobs.cdc.gov/
- [ ] Monitor CMS jobs:
  https://www.cms.gov/about-cms/work-with-us/careers
- [ ] Monitor JHU APL new-graduate roles:
  https://careers.jhuapl.edu/new-grads
- [ ] Monitor JHU APL AI/ML/Data Science roles:
  https://careers.jhuapl.edu/ai-ml-ds

## Phase 7: Interview Preparation

### Data Analyst
- [ ] Practice SQL (LeetCode, HackerRank, Mode Analytics)
- [ ] Practice Python data analysis (pandas, numpy)
- [ ] Prepare portfolio projects with healthcare datasets

### Consulting
- [ ] Practice case interviews:
  - PrepLounge: https://www.preplounge.com/
  - CaseBasix: https://www.casebasix.com/
- [ ] Practice market sizing and structured communication
- [ ] Read healthcare consulting case studies on firm websites

### Product Manager
- [ ] Practice product cases:
  - Exponent: https://www.tryexponent.com/
  - Product School: https://productschool.com/
- [ ] Write product teardowns of health tech products
- [ ] Study FDA regulatory pathways (510(k), PMA, SaMD, De Novo)
